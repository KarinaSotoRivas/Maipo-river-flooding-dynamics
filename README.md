This data cointains the inputs fo numerical simulations for Maipo River, where :

  hydrograph_TX_AA.npz is the hydrograph for X return period for hydrograph AA.\
  geo_maipo_bar.slf, geo_maipo_half_bar.slf, geo_maipo.slf are the domains for cases with or without bar.\
  t2d_case_AA.cas gai_erosion_AA.cas are the telemac 2d and gaia's steering file for the case AA. \
  geo_maipo.cli indicates the type of boundary conditions. \
  geo_maipo_AA.qsl are the boundary conditions timeseries.
